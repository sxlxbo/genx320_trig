为什么这个“合体脚本”最靠谱？
0.5秒的安全缓冲：在相机开始执行 log_raw_data 后，代码强制休眠了 0.5 秒。
这确保了当第一个 LED 亮起、第一个 Trig 脉冲发出时，相机的录像机早已经转动起来了。

一键式傻瓜操作：你只需要运行这一个 Python 文件，就能自动走完“开相机 -> 写 Bias -> 开始录像 -> 精准闪灯 -> 停止闪灯 -> 停止录像 -> 保存文件”的全流程。
这对做上百次重复实验的科研过程来说，极大地降低了心智负担。

数据百分百对齐：当你后期拿到 sync_test_rgb_01.raw 去解析时，读到的第 1 个 Trigger 绝对是 Red，第 2 个绝对是 Green，第 3 个绝对是 Blue。 
闭着眼睛套循环公式解算就行了。
