# 查看目录下的偏置
ls /usr/local/share/metavision/hal/resources/biases/

# 复制粘贴 genx320mp_CD_standard.bias

cp /usr/local/share/metavision/hal/resources/biases/ ~/偏置名.bias     表示从复制这个内置的偏执文件到pi目录下为新名字

# 然后是更改参数
nano 偏置名.bias

示例更改的是bias_diff_on为55，bias_diff_off为58

ctrl+o,保存，再按enter，再按ctrl+x退出nano编辑模式


# 使用更改后的新偏置

metavision_viewer -b 偏置名.bias
