# 1. 准备对焦目标
  在目标工作距离处，用显示器显示Prophesee 标准黑白闪烁方块（官网可下载，频率设为 10Hz），确保目标充满传感器视野。
# 2. 运行官方对焦命令,打开终端，直接输入：
  metavision_blinking_pattern_focus
  工具会自动识别 GenX320，实时输出清晰度分数（分数越高越清晰）
# 3. 硬件调焦（M12 镜头）
  握住 M12 镜头的对焦环，以1/32 圈为步长旋转：
    顺时针拧入：对焦距离变近。
    逆时针拧出：对焦距离变远。
  边旋转边观察终端的清晰度分数，找到分数的最大值并稳定，即为最佳对焦位置。

# 二、进阶优化命令（针对特定场景）
  锁定 ROI 区域对焦（排除背景干扰）
  若只需要对画面中心区域对焦，运行：
  
  metavision_blinking_pattern_focus --roi 100,100,200,200
  （参数说明：--roi x,y,w,h，x/y 为区域左上角坐标，w/h 为宽高）

##  指定闪烁频率（与显示器严格匹配）
    若显示器闪烁频率为 5Hz，运行：
    metavision_blinking_pattern_focus --frequency 5

#  三、（可选）验证对焦效果
  对焦完成后，可运行简易预览工具确认事件流清晰度：
    metavision_simple_viewer
